# Sales EDA Capstone Project

## Overview
This project performs end-to-end Exploratory Data Analysis (EDA) on a sales dataset using Python.

## Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- SciPy

## Project Workflow
- Data loading and cleaning
- Feature engineering
- Exploratory Data Analysis (EDA)
- Sales and profit analysis
- Correlation analysis
- ANOVA statistical testing

## Key Insights
- Analyzed sales performance across regions
- Identified profitable product categories
- Observed monthly sales trends
- Tested statistical differences in sales using ANOVA

## Conclusion
This project demonstrates practical data analysis skills suitable for fresher-level roles.
